🌙 Zikr e Falah

A beautifully designed Islamic Tasbeeh & Zikr counter application built with Flutter.
Zikr e Falah helps users perform daily Azkar with a clean interface, smooth counting experience, and customizable targets.

✨ Features
📿 Digital Tasbeeh Counter
🌙 Beautiful Islamic UI Design
🎯 Custom Target System
➕ Add Custom Zikr
✏️ Edit Target Counts
🔔 Vibration Feedback on Completion
✅ Completion Status Indicators
💾 Auto-save Counter Progress
🎨 Smooth Animations & Modern UI
📱 Responsive Flutter Design
